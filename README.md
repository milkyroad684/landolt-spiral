# Landolt Spiral 光学パズル

視力検査でおなじみのランドルト環 (C字マーク) を拡大縮小すると、切れ目が螺旋状に回転する — その一つの法則だけで成立する光学パズルです。

光をプリズム・ミラー・ハーフミラー・色フィルターで導き、外周の眼 (◎) に当ててください。**操作は環の大きさを変えるだけ**。

## あそぶ

[ここ](./index.html) を開く、または GitHub Pages の URL にアクセス。

## 仕様

- **単一HTMLファイル** (依存ライブラリなし、純 Vanilla JS + SVG)
- 全 10 ステージ、6 種類のギミック (N/P/M/H/L/F)
- ライト/ダーク両モード対応
- スマートフォンのタッチ操作にも対応

## ギミック凡例

| 表示 | 種類 | 動作 |
|---|---|---|
| グレー | 通常環 (N) | 本体は光を吸収、切れ目は通す |
| 琥珀 | プリズム環 (P) | 本体で固定角度の屈折、切れ目は通す |
| 青 | ミラー環 (M) | 本体で正反射、切れ目は通す |
| ピンク | ハーフミラー環 (H) | 本体で透過と反射に分岐 |
| 琥珀点線 | 可変レンズ (L) | 屈折角がサイズに反比例 |
| 緑・赤 | フィルター (F) | 本体は色一致時のみ通過、切れ目は素通り |
